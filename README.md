# Åpen kildekode-alternativer til proprietær programvare

En enkel oversikt over åpen kildekode-alternativer til proprietær programvare.

All data ligger i én fil: `src/data/alternatives.json`. Nettsiden bygges automatisk fra denne filen. Legger du til noe der, dukker det opp på siden.

## Kjør lokalt

```
npm install
npm run dev
```

Siden kjører på `localhost:4321`.

## Bygg

```
npm run build
```

## Bidra

Rediger `src/data/alternatives.json` og lag en PR. Vær vennlig å beskriv programvaren du legger til, og hvorfor du mener det er et godt alternativ. Det må være 100% åpen kildekode, og det må være et reelt alternativ til den proprietære programvaren.

## Teknologi

Bygget med [Astro](https://astro.build).
